HITL Customer Support Agent

A LangGraph support agent that drafts replies with an LLM but pauses for a human on Slack whenever the stakes are real — refunds, angry customers, policy edge cases.

🏆 Flagship · Agentic
⚙️ How it's built
  • LangGraph orchestration — 15 nodes, 5 conditional edges, durable pause/resume via SQLite checkpointer (survives process restarts)
  • 3 capability-isolated MCP servers (Read / Email-Write / Slack-Write) — bounded blast radius against prompt injection
  • Real Gmail IMAP+SMTP & real Slack approval cards (Approve / Edit / Reject) — no mocks on the I/O layer
  • Two-gate safety routing + a v4 multi-agent upgrade (Researcher + Drafter↔Critic loop)
  • PII redaction at entry, restored at finalize — round-trip identity tested
📊 Results & rigor
  • 0% false-auto-send rate — the primary safety metric, test-asserted & machine-checked
  • 148 / 148 tests passing across both architecture modes
  • Full observability — LangSmith traces, Prometheus metrics, live Grafana dashboard via docker compose
  • STRIDE threat model + CI (ruff, mypy, pytest, pip-audit, bandit)
  • Honest, reproducible eval — head-to-head v3-vs-v4 across curated + Bitext datasets
